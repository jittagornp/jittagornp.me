เป้าหมายในชีวิต

เป็นโปรแกรมเมอร์ ที่สร้างผลงานเจ๋งๆ ระดับโลกให้คนทั่วโลกใช้ และมีบริษัทซอฟต์แวร์เป็นของตัวเอง

สภาพแวดล้อมการทำงานที่ชอบ

  • เป็นองค์กรที่มีผลิตภัณฑ์ (product) เป็นของตัวเอง
  • เป็นองค์กรที่อยากสร้างผลิตภัณฑ์เจ๋งๆ ส่งไปยังผู้ใช้
  • มีเอกลักษณ์/วัฒนธรรมองค์กรเป็นของตัวเอง
  • ไม่มีลำดับชั้นของพนักงาน มีแค่ตำแหน่งที่ใช้ทำงาน
  • ไม่ใส่สูทผูกไท้ ไปทำงาน
  • เรื่องงานถึงไหนถึงกัน
  • มี Best Practices ในการทำงาน
